# Hume — Curious People LMS

> A full-featured Learning Management System for [hume.curiouspeople.co.za](https://hume.curiouspeople.co.za)

Built on Laravel 10, this platform provides course creation, student enrollment, quizzes, certificates, subscriptions, and a full admin dashboard.

---

## Technology Stack

| Layer | Technology |
|---|---|
| Backend | PHP 8.1+, Laravel 10.21 |
| Frontend | Blade templates, Bootstrap 5.1.3 |
| Database | MySQL 5.7+ |
| Storage | Local + DigitalOcean Spaces (S3-compatible CDN) |
| Auth | Laravel Sanctum, Socialite (Google, Facebook, Twitter, LinkedIn) |
| Payments | Stripe |
| Build | Laravel Mix, Webpack, Node.js |

## Quick Start (Local Development)

### Prerequisites

- PHP 8.1+
- Composer
- Node.js 16+
- MySQL 5.7+

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/chrisexpertai/hume-curiouspeople.git
cd hume-curiouspeople

# 2. Install PHP dependencies
composer install

# 3. Install Node dependencies
npm install

# 4. Set up environment
cp .env.example .env
php artisan key:generate

# 5. Configure your database in .env, then run migrations
php artisan migrate
php artisan db:seed

# 6. Create storage symlink
php artisan storage:link

# 7. Compile assets
npm run dev

# 8. Start the development server
php artisan serve
```

### Default Admin Login

```
Email:    admin@example.com
Password: 123
```

> **Change this immediately on any live or staging environment.**

## Key Features

- **Courses** — Sections, lessons, video lectures, file attachments
- **Assessments** — Quizzes with multiple question types, assignment submissions
- **Certificates** — Auto-generated PDF certificates on course completion
- **Subscriptions** — Flexible plans with Stripe payment processing
- **Instructors** — Application workflow, earnings tracking, multi-instructor courses
- **Students** — Enrollment, progress tracking, discussion forums
- **Blog** — Post management with categories and comments
- **Admin** — Full dashboard: users, courses, payments, tickets, analytics
- **Plugins** — Certificate, MultiInstructor, StudentsProgress (extensible architecture)
- **Social Login** — Google, Facebook, Twitter, LinkedIn
- **Media Library** — Laravel File Manager with S3/local storage
- **Multi-language** — Language switching support

## Environment Variables

Copy `.env.example` to `.env` and fill in:

| Variable | Description |
|---|---|
| `APP_KEY` | Generated by `php artisan key:generate` |
| `APP_URL` | Your full domain (e.g. `https://hume.curiouspeople.co.za`) |
| `DB_HOST / DB_DATABASE / DB_USERNAME / DB_PASSWORD` | MySQL credentials |
| `AWS_ACCESS_KEY_ID / AWS_SECRET_ACCESS_KEY` | DigitalOcean Spaces keys |
| `AWS_BUCKET / AWS_URL` | Spaces bucket and CDN URL |
| `STRIPE_KEY / STRIPE_SECRET` | Stripe payment keys |
| `MAIL_*` | SMTP mail configuration |
